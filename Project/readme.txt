This is the main working repository for the Kaggle competition. It has the following
structure:

1) Data: In this folder both the raw and the transformed data are stored
2) Literature and Sources: This folder contains relevant publications 
3) Work: Main working directory in this repository. It has the following structure:
		3.1) Output: Here all prediction outputs for the different models are 
			     saved.
		3.2) R_Scripts: Contains all R-Files. The files have the following
				identifications (besides their names):
				- F_ --> File
				- First digit 0: Data related file 
				- Frist digit 1: Model related file
				- Second digit: Running number
